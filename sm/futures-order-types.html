Two-Stage Trailing Stop (Advanced Algo Concept)
Your idea: loose ₹10 trail before the target, switch to tight ₹5 trail beyond the target to catch extended rallies. No retail UI supports this — requires Dhan API / Python algo.

Python Pseudocode — Two-Stage Trailing via Dhan API

# Entry: ₹100 | Target: ₹130 | Initial SL: ₹90
# Phase 1: ₹10 trail until price exceeds target
# Phase 2: ₹5 trail beyond target — catches extended rally

entry_price   = 100
target_price  = 130
current_sl    = 90
highest_price = entry_price

while position_is_open:
    current_price = get_ltp()

    if current_price > highest_price:
        highest_price = current_price

        if current_price <= target_price:
            trail_gap = 10   # Phase 1 — loose trail
        else:
            trail_gap = 5    # Phase 2 — tight trail beyond target

        new_sl = highest_price - trail_gap

        if new_sl > current_sl:
            current_sl = new_sl
            update_stop_loss(current_sl)  # Dhan API call

    if current_price <= current_sl:
        exit_position()   # Market order exit
        break
